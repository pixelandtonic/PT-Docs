``{exp:playa:coparent_ids}``
===============================

You can use the ``{exp:playa:coparent_ids}`` tag pair to get the entry IDs that share a common child with other entries.

``{exp:playa:coparent_ids}`` supports almost all of ``{exp:channel:entries}``’ `tag parameters <http://ellislab.com/expressionengine/user-guide/modules/channel/channel_entries.html#parameters>`_, as well as the following:

.. include:: /templating/_includes/shared_coparent_params.rst

``delimiter``
	The string used to separate the returned entry IDs. (The default is “``|``”.)
