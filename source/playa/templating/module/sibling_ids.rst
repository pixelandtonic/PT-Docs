``{exp:playa:total_siblings}``
==============================

Tag Parameters
--------------

``{exp:playa:total_siblings}`` supports almost all of ``{exp:channel:entries}``’ `tag parameters <http://ellislab.com/expressionengine/user-guide/modules/channel/channel_entries.html#parameters>`_, as well as the following:

.. include:: /templating/_includes/shared_sibling_params.rst

``delimiter``
	The string used to separate the returned entry IDs. (The default is “``|``”.)
